# Where Next: Denver Metro

Interactive research tool for Denver-metro relocation: 40 neighborhoods ranked live across
14 weighted criteria, 63 ranked schools, a real map, an onboarding quiz, and shareable
weight links. One HTML file; open it, or serve it on GitHub Pages.

- `index.html` — the entire app (v4.3)
- `CLAUDE.md` — working agreement + invariants for Claude Code
- `test/verify.mjs` — headless verification suite (`npm test`)
- `data/denver-hoods.geojson` — official Denver polygon source
- `docs/DECISIONS.md` — decision log, provenance, backlog
